SECTION 1

Logging In

Use your JCB machine number and password to sign in. Your credentials are provided by your admin or supervisor.

Smart Lekka Login Screen

Steps to Log In

  1. Open Smart Lekka in your phone browser The login screen appears automatically.
  2. Enter your JCB Number Type your JCB machine number in the "JCB Number / Mobile" field (e.g. JCB001).
  3. Enter your Password Type the password given to you by your admin.
  4. Tap Sign In You are taken to your personal Dashboard.
💡 Your JCB Number is your login ID. It is usually in the format JCB001, JCB002, etc. Contact your admin if you need your credentials.

SECTION 3

Tracking a Live Work Session

Use the live timer when you start working in the field. The app tracks the exact time automatically.

Starting a Session

  1. Tap "Track" in the bottom navigation The Track page opens, showing the Timer tab by default.
  2. Select or add a Worker Choose the worker from the dropdown, or tick "Add new worker on the fly" to create one instantly.
  3. Enter the Field Name Type the name of the field or location (e.g. "North Pasture", "Village Road").
  4. Tap "START TRACKING" The live timer begins counting. You can safely close the app — the session stays active.
03:24:15
Session In Progress

Stopping a Session

  1. Return to the Track page The active session is shown with the elapsed time.
  2. Tap "Stop" A review screen appears showing total hours worked and the automatically calculated payment amount.
  3. Review and Save Check the details and tap "Save Work Session" to save it permanently.
💡 Auto-Calculated Payment The payment amount is calculated as: Hours × Rate per Hour. The current rate is set by your admin and shown below the amount.
SECTION 4

Adding a Manual Work Entry

If you forgot to start the timer or need to record a past session, use Manual Entry to log the specific date and times.

Manual Entry Form — Track Page
  1. Tap "Track" → Switch to "Manual Entry" tab Tap the "Manual Entry" tab at the top of the Track page.
  2. Select or add a Worker Choose from the dropdown or add a new worker.
  3. Enter the Field Name Type the location or field where the work was done.
  4. Set the Date Tap the date field and select the date when the work was done.
  5. Set Start Time Use the hour and minute dropdowns, then tap AM or PM. Example: 8 : 00 AM.
  6. Set End Time Same as Start Time. Example: 5 : 00 PM.
  7. Check the Payment Amount It is auto-calculated from the hours and rate. You can edit it if needed.
  8. Tap "Save Work Entry" The session is saved and appears in History.
💡 Start Time & End Time on Mobile On a phone, the Start and End time pickers appear stacked (one above the other) so they are easier to tap. On a tablet or desktop they appear side by side.
⚠️ End Time must be after Start Time If the end time is earlier than the start time, the calculated hours will be zero or negative. Always double-check your times.
SECTION 5

Viewing Work History

The History page shows all your logged work sessions and all payments made to each worker.

How to Use Filters

  • Worker Filter — Choose a specific worker to see only their sessions.
  • Status: Pending — Show only workers who still have unpaid balances.
  • Status: Settled — Show only workers who have been fully paid.
  • Status: All — Show all workers regardless of payment status.

SECTION 6

Adding Expenses

Log any JCB-related costs such as fuel, repairs, or materials. These are tracked separately from worker wages.

How to Add an Expense

  1. Tap "Expenses" in the bottom navigation The Expenses page opens showing this month's expenses by default.
  2. Tap the "+ Add" button A form slides in from the bottom.
  3. Enter a Description Briefly describe the expense (e.g. "Diesel fill-up", "Oil change").
  4. Select a Category Choose from Fuel, Maintenance, Salary, Materials, or Other.
  5. Enter the Amount (₹) Type the total cost of the expense.
  6. Set the Date Defaults to today — change if it was an earlier expense.
  7. Tap "Add Expense" The expense is saved and appears in the list.

Marking an Expense as Paid

Each expense can be partially or fully marked as paid. Tap the expense, then tap "Record Payment" and enter the amount paid. The pending amount updates automatically.

Filtering by Date

Use the From and To date fields to see expenses from a specific period. Tap "Show All Time" to remove the date filter.

SECTION 7

Downloading Your Report

Generate a PDF report of your work sessions and payments for any time period.

  1. Go to the Dashboard The report is generated from your Dashboard data.
  2. Tap "Download Report" The report button appears in the top-right when you have workers with data.
  3. Select Workers and Period Choose "All Workers" or a specific worker. Select the time period (This Month, Last Month, Custom, or All Time).
  4. Tap "Generate PDF" The PDF downloads to your device. It includes your JCB machine name in the report header.
💡 Your JCB name is shown in the PDF Since you are logged in as a JCB operator, the downloaded report automatically includes your JCB number/name in the filter summary.

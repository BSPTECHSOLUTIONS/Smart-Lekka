🏗️ Part A — Supervisor Role

Supervisors manage one client's JCB fleet, view aggregated or per-JCB data, record worker payments, and download reports.

A1

Logging In as a Supervisor

Your mobile number is your login ID. Your admin provides your password.

Login Screen (mobile)
  1. Open Smart Lekka in your browserThe login screen appears.
  2. Enter your Mobile NumberYour 10-digit mobile number in the "JCB Number / Mobile" field.
  3. Enter your PasswordGiven to you by the admin.
  4. Tap Sign InYou land on the Dashboard.

A3

Filtering by JCB Machine

The blue JCB filter bar lets you focus on one machine's data. It appears on Dashboard, History, and Expenses.

JCB Filter Bar (Supervisor view)
▼ All JCBs ▾
  • All JCBs — Shows combined data from all machines under your client.
  • JCB-001 / JCB-002 … — Filters to show only that machine's workers and work sessions.
💡 Filter is independent per page Changing the filter on the Dashboard does not affect the Expenses or History pages — each page has its own filter.

How to Record a Payment

  1. Find the worker card on the DashboardScroll the pending workers list.
  2. Tap the "Pay" buttonA dialog opens with the current pending amount pre-filled.
  3. Enter the amount paidCan be partial or the full pending amount.
  4. Tap "Record Payment"The balance updates immediately. Full payment moves the worker off the home screen.
⚠️ Payments cannot be undone Double-check the amount before saving. Contact the admin if a correction is needed.

A7

Downloading PDF Reports (Supervisor)

Generate a professional PDF covering work sessions and payments for any period and any JCB filter.

  1. Set the JCB filter on the DashboardChoose "All JCBs" or a specific machine before opening the report dialog.
  2. Tap "Download Report"Top-right of the Dashboard (only visible when workers exist).
  3. Choose Workers and PeriodSelect a worker filter and a date range (This Month / Last Month / Custom / All Time).
  4. Check the summary previewThe "Report will include" box shows the JCB filter name if one is active.
  5. Tap "Generate PDF"The PDF downloads automatically. It includes sessions, payments, and totals per worker.

B3

Tracking a Live Work Session

Use the live timer to track work in real time. Start it when work begins, stop it when work ends.

  1. Tap "Track" in the bottom barOpens the Track page on the Timer tab.
  2. Select or add a WorkerChoose from the dropdown or tick "Add new worker on the fly".
  3. Enter the Field Namee.g. "North Pasture", "Main Road Site".
  4. Tap "START TRACKING"The timer begins. You can close the app safely — the session stays active.
03:24:15
Session Running

Stopping the Session

  1. Return to the Track pageThe active timer is shown.
  2. Tap "Stop"A review screen shows total hours and the calculated payment amount.
  3. Tap "Save Work Session"Saved permanently and added to History.
💡 Auto-calculated Payment Amount = Hours Worked × Rate per Hour. The rate is set by your admin and shown below the amount field.
B4

Adding a Manual Work Entry

Use Manual Entry to log a past work session when you forgot to use the live timer.

Manual Entry Form — Track Page
  1. Tap "Track" → tap "Manual Entry" tabThe manual form opens.
  2. Select the WorkerChoose from the dropdown or add a new worker.
  3. Enter the Field NameWhere the work was done.
  4. Set the DatePick the date when the work happened.
  5. Set Start TimeChoose hour, minute, and AM/PM. On mobile these appear stacked for easy tapping.
  6. Set End TimeSame as Start Time. Must be later than start time.
  7. Check the AmountAuto-calculated. You can edit it if needed.
  8. Tap "Save Work Entry"The session is saved and appears in History.
⚠️ End Time must be after Start Time If end time is earlier than start time, the calculated hours will be wrong. Always double-check both times.

B6

Adding Expenses (JCB Operator)

Log any cost related to your JCB — fuel, maintenance, materials, etc.

  1. Tap "Expenses" in the bottom barShows this month's expenses by default.
  2. Tap "+ Add"The add expense form opens.
  3. Enter a Descriptione.g. "Diesel fill-up 40L".
  4. Select a CategoryFuel / Maintenance / Salary / Materials / Other.
  5. Enter the Amount (₹)The total cost of this expense.
  6. Set the DateDefaults to today. Change if it was an earlier date.
  7. Tap "Add Expense"Saved and shows in the list immediately.
B7

Downloading Your PDF Report

Get a PDF of your work sessions and worker payments from the Dashboard.

  1. Go to the Dashboard (Home)Your data is shown automatically.
  2. Tap "Download Report"Button in the top-right (appears when you have worker data).
  3. Choose Workers and Date RangeSelect all or a specific worker. Choose the period.
  4. Tap "Generate PDF"Downloads to your device. Your JCB name/number is shown in the PDF header automatically.

SECTION 1

Logging In

Use your mobile number and password provided by your admin to sign in as a Supervisor.

Smart Lekka Login Screen

Steps to Log In

  1. Open the Smart Lekka app Go to the app URL in your phone's browser. The login screen appears automatically.
  2. Enter your Mobile Number Type your 10-digit mobile number in the "JCB Number / Mobile" field.
  3. Enter your Password Type the password given to you by the admin.
  4. Tap Sign In You will be taken to the Dashboard.
💡 Supervisor Credentials Format Your mobile number is your login ID. If you forget your password, ask your Admin to reset it from the Admin panel.

SECTION 3

Filtering by JCB Machine

As a Supervisor, you oversee multiple JCB machines. Use the JCB filter to view data for one specific JCB at a time.

Where to Find the Filter

Just below the Dashboard title, you will see a blue filter bar with a dropdown:

JCB Filter Bar (appears for Supervisors only)
▼ Filter All JCBs ▾

How to Filter

  1. Tap the filter dropdown It shows "All JCBs" by default — this shows data from all machines combined.
  2. Select a JCB machine Choose a specific JCB (e.g. "JCB-001") from the list. The worker list and summary will update to show only work done by that JCB.
  3. Switch back to All JCBs Select "All JCBs" to return to the combined view.
💡 The same JCB filter is available on the Expenses page and the History page. Each page has its own independent filter so you can compare across pages.

SECTION 5

Recording a Payment

When you pay a worker, record it in Smart Lekka so the pending balance stays accurate.

  1. Find the worker on the Dashboard Scroll through the pending workers list.
  2. Tap the "Pay" button A payment dialog opens showing the worker's current pending amount.
  3. Enter the amount paid Type the exact amount you are paying today (can be partial or full).
  4. Tap "Record Payment" The pending balance updates immediately. If fully paid, the worker moves off the home screen.
💡 Partial Payments are OK You can record multiple partial payments over time. Each payment is logged with a date so there is a full audit trail.
⚠️ Cannot Undo Payments Once a payment is recorded it is permanent. Double-check the amount before saving.

SECTION 7

Expenses Page

Track all JCB-related expenses such as fuel, maintenance, salaries, materials, and other costs.

JCB Filter on Expenses

As a Supervisor, you see expenses from all JCB machines in your client by default. Use the blue JCB filter bar to narrow down to a specific machine.

Date Range Filter

Set a From and To date to see expenses within a specific period. Tap "Show All Time" to remove the date filter.

SECTION 8

Downloading PDF Reports

Generate a professional PDF report from the Dashboard for any time period, for all or selected workers, filtered by JCB if needed.

  1. Go to the Dashboard Make sure the JCB filter is set to the machine you want (or "All JCBs").
  2. Tap the "Download Report" button It appears in the top-right of the Dashboard. If there are no workers it will not appear.
  3. Choose Workers Select "All Workers" or pick a specific worker from the dropdown.
  4. Choose a Date Period Options: All Time, This Month, Last Month, or a Custom date range.
  5. Review the Summary The "Report will include" section shows exactly what will be in the PDF. If a JCB filter is active, it shows the JCB name in blue.
  6. Tap "Generate PDF" The PDF downloads to your device automatically.
💡 What's in the PDF? The PDF includes: a summary table (earned, paid, pending per worker), all individual work sessions with dates and hours, and all payment records.
SECTION 9

Navigation Guide

The app has a bottom navigation bar (on mobile) with the following pages:

🏠 Home
⏱ Track
📋 History
💰 Expenses
  • Home (Dashboard) — Overview summary + pending workers + JCB filter.

  • History — Full session and payment history with JCB and worker filters.
